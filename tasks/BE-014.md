# BE-014: SMTP інтеграція та HTML-шаблони листів

Фаза: 1 (MVP)
Тип: Backend

## Мета
Забезпечити відправку email через SMTP з використанням HTML-шаблонів для всіх типів нотифікацій.

## Скоуп
- Параметри SMTP з .env (SMTP_HOST/PORT/USER/PASS/TLS)
- Шаблони HTML для:
  - NEW_CASE, STATUS_CHANGED, COMMENT_PUBLIC, COMMENT_INTERNAL, REASSIGNED, ESCALATION
- Сервіс відправки листів (викликається з Celery-таски)

## Визначення готовності (DoD)
- Листи відправляються для всіх сценаріїв з коректними темами/контентом
- Помилки відправки логуються (і ретраяться в BE-013)

## Тести
- Смок-тести відправки з моканим SMTP; перевірка тем/шаблонів

## Залежності
- BE-013

## Позначки
- SMTP, Templates, Emails
