# FE-008: Адмін розділ — користувачі

Фаза: 1 (MVP)
Тип: Frontend

## Мета
Інтерфейс для перегляду/створення/редагування користувачів, деактивації та скидання пароля.

## Скоуп
- Таблиця користувачів: ПІБ, Логін, Email, Роль, Категорії, Статус
- Фільтри за роллю/статусом; сортування
- Форми створення/редагування з валідаціями; дії Deactivate/Reset Password

## Визначення готовності (DoD)
- Операції працюють через API, помилки відображаються

## Тести
- Створення/редагування/деактивація; обмеження за ролями

## Залежності
- BE-012

## Позначки
- Admin, Users, Forms
