# INF-001: docker-compose для API/Worker/Beat/Redis/DB/FE/Nginx

Фаза: 1 (MVP)
Тип: Infra/DevOps

## Мета
Створити docker-compose конфіг із сервісами бекенду, фронтенду, Redis, PostgreSQL та Nginx (prod).

## Скоуп
- Сервіси: api (uvicorn), worker (celery), beat (celery beat), redis, db (postgres), frontend (next), nginx
- Мережі, томи (db-data, media, static)
- Базові healthcheck’и

## Визначення готовності (DoD)
- `docker compose up` піднімає всі сервіси локально

## Тести
- Smoke: доступність API/FE через nginx (prod-профіль)

## Залежності
- BE-* , FE-001

## Позначки
- Docker, Compose
